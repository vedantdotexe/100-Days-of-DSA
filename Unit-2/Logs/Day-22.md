### 🟢 Day 22: March 31, 2026
**Topic:** Longest Common Prefix (Vertical Scanning)
**Language:** Java ☕
**Vibe:** 📏 Measuring the match

Today I solved the Longest Common Prefix problem. I used "Vertical Scanning," which compares the same index across all strings in the array. It's efficient because as soon as one character doesn't match, we stop. This was a great exercise in using `substring()` and nested loops in Java.
