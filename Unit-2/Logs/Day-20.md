**Language:** Java ☕
**Vibe:** 🔄 Back in the Game!

Picked up the challenge again after a break. Today I learned a "cheat code" for string rotations. Instead of manually shifting characters, if you add a string to itself (s1 + s1), it contains every possible rotation of that string. 

It’s a clever O(n) trick that simplifies a problem that looks hard at first glance. Good to be back at it.
