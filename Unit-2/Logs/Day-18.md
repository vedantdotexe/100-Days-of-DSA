### 🟢 Day 18: March 15, 2026
**Topic:** First Unique Character (Two-Pass Array)
**Language:** Java ☕
**Vibe:** 🔎 Searching for the One

Today I solved the First Unique Character problem. I used a "Two-Pass" approach. The first loop counts the characters, and the second loop checks the string against those counts. 

This is a great example of why we sometimes need to look at the data twice. The time complexity is still O(n) because O(2n) simplifies to O(n), and the space is O(1) because our tally sheet is always size 26.
