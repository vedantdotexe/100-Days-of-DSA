### 🟢 Day 21: March 30, 2026
**Topic:** String Compression (StringBuilder)
**Language:** Java ☕
**Vibe:** 📦 Making things compact

Today I worked on String Compression. I learned that in Java, using `String += "more text"` is slow because strings are immutable. Instead, I used `StringBuilder`, which is like a dynamic array for characters.

The logic involves a single loop that counts consecutive characters and appends them to the builder. It’s an O(n) solution that taught me about memory efficiency in Java.
