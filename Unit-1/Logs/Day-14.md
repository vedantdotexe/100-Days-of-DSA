### 🟢 Day 14: March 12, 2026
**Topic:** Subarray Sums (Sliding Window Intro)
**Vibe:** 🖼️ Moving the Frame

Today I started looking at "Subarrays"—which are just chunks of an array that stay together. I wrote a script to find the sum of every 3 numbers in a row.

The main takeaway was learning where to stop the loop. If the window is size 3, the loop has to stop 2 spots before the end of the array so the "frame" doesn't fall off the edge. It’s a nested loop for now, which isn't the fastest way to do it, but it helped me visualize how a "window" of data moves across a larger set.
