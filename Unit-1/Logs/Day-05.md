### 🟢 Day 5: March 2, 2026
**Topic:** Removing Duplicates (Two-Pointer Technique)
**Vibe:** 🧹 Cleaning the Track

Today was all about data optimization. I tackled the problem of removing duplicates from a sorted array "In-Place." After learning the Two-Pointer method for reversal yesterday, today showed me how those same pointers can be used for "filtering" data.

The "Aha!" moment was realizing that since the array is sorted, I only need to compare the current element with the last unique one I found. I used a "Gatekeeper and Scout" logic: the Gatekeeper (i) only moves forward when the Scout (j) finds a brand-new value. This keeps the Space Complexity at O(1), which is a huge performance win. It’s starting to click how these simple pointer movements can solve real-world data-cleaning problems in the MERN stack.
