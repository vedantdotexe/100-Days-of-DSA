### 🟢 Day 13: March 12, 2026
**Topic:** Array Intersection (Hash Set Logic)
**Vibe:** 🤝 Finding Common Ground

Today’s problem was about finding common elements between two arrays. Instead of using nested loops (which would be O(n*m)), I used a "Checklist" strategy with a JavaScript Object.

The "Aha!" moment was realizing that once I found a match, I needed to mark it as 'false' on my checklist to avoid duplicates in the result. This is a very efficient way to compare two datasets, which is something I’ll definitely use in the MERN stack when comparing user permissions or filtering product categories.
