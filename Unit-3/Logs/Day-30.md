Hit the 30-day milestone today! Started Sorting with Bubble Sort. It’s an $O(n^2)$ algorithm, so it’s not the most efficient, but the logic is very clear. I implemented an optimization using a `swapped` boolean—if the array becomes sorted mid-way, the algorithm stops early. It’s a great lesson in how nested loops interact with data.
