### 🟢 Day 34: May 2, 2026
**Topic:** Quick Sort
**Language:** Java ☕
**Vibe:** ⚡ Pivot Power

Today I tackled Quick Sort. Unlike Merge Sort, this doesn't need extra arrays (it's "in-place"), which makes it very memory efficient. The core logic is the 'Partition'—picking a pivot and organizing everything smaller to the left and larger to the right. It’s a bit more complex to code than Merge Sort, but it's often faster in the real world.
