### 🟢 Day 24: April 2, 2026
**Topic:** Binary Search (Iterative)
**Language:** Java ☕
**Vibe:** ⚡ Efficiency Boost

Started Unit 3 today with Binary Search. The jump from $O(n)$ (Linear Search) to $O(\log n)$ is massive. I learned why we use `mid = left + (right - left) / 2` instead of `(left + right) / 2`—it's to prevent integer overflow in Java if the array is huge. 

Binary search only works on sorted arrays, but when it works, it's incredibly fast.
