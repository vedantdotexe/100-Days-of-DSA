### 🟢 Day 28: April 6, 2026
**Topic:** Search in Rotated Sorted Array
**Language:** Java ☕
**Vibe:** 🎡 Dealing with the Twist

Today I tackled the Rotated Sorted Array problem. The key takeaway was that even when an array is rotated, Binary Search still works if you can identify the "sorted half." This taught me to look for properties of the data (like which side is increasing) rather than just assuming the whole thing is perfect. It's still O(log n), but the logic is much more clever.
