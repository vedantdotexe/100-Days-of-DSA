### 🟢 Day 32: April 10, 2026
**Topic:** Insertion Sort
**Language:** Java ☕
**Vibe:** 🃏 Sorting like a Pro

Today I finished the basic sorting algorithms with Insertion Sort. It’s much more "stable" and efficient for small or nearly sorted datasets than Bubble or Selection sort. I liked the logic of "shifting" elements instead of just swapping them—it feels more like how we handle physical objects. $O(n^2)$ in the worst case, but $O(n)$ if the array is already sorted!
