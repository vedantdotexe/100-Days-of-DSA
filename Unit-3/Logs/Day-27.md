### 🟢 Day 27: April 5, 2026
**Topic:** Find First and Last Position (Modified Binary Search)
**Language:** Java ☕
**Vibe:** 📍 Pinpointing Boundaries

Today I learned how to modify Binary Search to handle duplicates. Instead of stopping as soon as the target is found, I used a boolean flag to decide whether to keep searching on the left or right side. This effectively "squeezes" the search space to find the exact boundaries of a repeating number.
