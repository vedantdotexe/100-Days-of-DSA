### 🟢 Day 31: April 9, 2026
**Topic:** Selection Sort
**Language:** Java ☕
**Vibe:** 🔎 Searching for the Minimum

Today I implemented Selection Sort. Unlike Bubble Sort, which does many swaps per pass, Selection Sort only does one swap per pass. It "selects" the smallest element and puts it in its place. Even though the time complexity is still O(n²), it’s interesting to see how it minimizes the number of swap operations.
