### 🟢 Day 29: April 7, 2026
**Topic:** Find Peak Element
**Language:** Java ☕
**Vibe:** 🏔️ Mountain Climbing Logic

Today I learned that Binary Search isn't just for sorted arrays. You can use it on any data where you can determine which direction to move. By comparing `mid` to its neighbor, I could figure out if I was on an "upward slope" or "downward slope" to find a peak in O(log n) time. This is a big shift in how I think about search algorithms!
